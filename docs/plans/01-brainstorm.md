# Brainstorm — 가치 카드 우선순위 게임

> brainstorming skill 적용 결과. 원격 무인 세션이므로 AI 자체 판단으로 단일 패스 정리.

## 꼭 필요한 기능 (MUST)

1. **10장 가치 카드 랭킹 단계** — 드래그-앤-드롭(데스크탑) / 탭-스왑(모바일·키보드) 두 방법 모두 지원.
   - 카드: 정직 / 우정 / 자유 / 책임 / 정의 / 배려 / 성실 / 존중 / 협력 / 용기 (초등 도덕 핵심 가치 + 학년 수준).
   - 각 카드는 가치명 + 1줄 설명 (예: "정직 — 거짓 없이 사실 그대로 말함").
   - 항상 1~10위까지 정렬 상태가 화면에 보임.

2. **단계별 압축 — 10→5→3** — "다음 단계" 버튼으로 진행.
   - 10→5: 상위 5개 자동 선택 + "왜 이 5개?" 한 줄 메모 입력 (선택사항).
   - 5→3: 상위 3개 자동 선택 + 1순위 강조 + 최종 메모 입력 (선택사항).
   - 각 단계 진입시 "이전 단계로" 버튼으로 되돌리기 가능.

3. **결과 PNG 출력** — Canvas API로 최종 결과 카드 1장 다운로드.
   - 포함: TOP 3 가치 + 1순위 가치의 한 줄 메모 + 날짜.
   - **이름·번호 등 식별 정보 자동 제거** (원본 배제 기능 충실 이행).
   - 모둠 토의에서 화면에 띄우거나 출력해서 비교용으로 사용.

## 있으면 좋은 기능 (SHOULD)

1. **키보드 접근성** — Tab으로 카드 포커스, ↑/↓로 순위 이동, Space로 "고정" 토글.
   - 1:1 환경에서 마우스 없는 학생/태블릿 키보드 사용자 배려.

2. **"가치 설명 보기" 토글** — 카드를 한 번 클릭하면 가치 설명을 펼쳐 보고 결정 도와줌.
   - 도덕 교과 학습 보조 — 학생이 단어만 보고 직관 정렬하지 않도록.

## 절대 금지 기능 (MUST NOT)

1. **외부 공유·전송** — Twitter share / 카카오톡 share / URL 단축 등 일체 없음.
   - 원본 명세 "결과 외부 공유 ✕" 직접 인용. PNG는 학생 로컬 다운로드만.

2. **학생 식별 정보 입력 필드** — 이름·반·번호·생년월일 등 입력 칸 자체를 두지 않음.
   - 원본 "학생 식별 결합 ✕". 결과 PNG에도 학생 식별자 자동 제거.

3. **점수·랭킹·정답 표시** — "이게 정답입니다" 같은 결과 판정 또는 학생 간 비교 점수.
   - 가치관 탐색은 정답이 없음. 결과를 평가·등수화하면 교육 목적 훼손.

4. **외부 네트워크 호출** — 모든 동작 100% 오프라인. 폰트도 시스템 폰트 fallback 우선.
   - GH Pages 정적 호스팅 + CDN 의존 0 → 학교 망 차단 환경에서도 100% 동작.
